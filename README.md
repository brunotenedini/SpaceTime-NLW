## 🖥️ Projeto 
Esse é um projeto Web Responsivo de uma cápsula do tempo para exibir memórias em uma time-line

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante a NLW da RocketSeat com as seguintes tecnologias:

- HTML
- CSS
- Git e GitHub
